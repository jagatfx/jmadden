Virtual Reality Hackathon

17 Apr 2016 on Virtual-reality, Hackathon, Content

I was fortunate enough to be able to participate in the Tech Valley AR/VR Gaming Hackathon in Troy, New York. It was incredible to be able to experiment with the various virtual reality technologies like the Oculus Rift, the HTC Vive and Leap Motion controller. I made an entry with the Unity game engine called RalphVR that is a recreation of the handheld electronic memory game simon in virtual reality. My entry which won Best Game is documented here.
